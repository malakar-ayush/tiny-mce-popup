A simple plugin that helps you add a pop up to your tiny mce

== Description ==

This plugin comprises of a simple way to add pop to your tiny mce. You can see a button called Add pop up in the visual editor. You can use that to add new pop up content to your wordpress powered content





== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/add-popup-to-tiny-mce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
